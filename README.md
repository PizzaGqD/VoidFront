# VoidFront

Браузерная sci-fi RTS / tactical arena game.

Текущая основная ветка репозитория: `cleanup/single-yagames-release`.
Это самая свежая ветка под одиночный релиз и Яндекс Игры.

## Что в репозитории

- `cleanup/single-yagames-release`  
  Актуальная ветка под single-player релиз, чистку legacy-кода, ботов и подготовку билда для Яндекс Игр.

- `master`  
  Историческая основная ветка проекта до переноса фокуса на single-player release.

- `dev`  
  Общая рабочая ветка разработки до поздних релизных чисток.

- `dev-network`  
  Сетевые и мультиплеерные эксперименты.

- `dev-network-refactor`  
  Более глубокий рефактор сетевого слоя и authority-пайплайна.

- `local-zbt-4p-core`  
  Локальное ядро под 4 игроков и связанные боевые/коридорные эксперименты.

- `refactor/io-release-foundation`  
  Базовая ветка под релизный рефактор и подготовку структуры проекта.

- `new-design`  
  Ветка с UI/UX и визуальными редизайн-экспериментами.

- `VPS-release-v0.1`  
  Подготовка self-hosted / VPS-мультиплеерного релиза.

- `COSMIC_REBALANCE`  
  Отдельная ветка под ребаланс экономики, боёвки и темпа матча.

- `ALPHA`  
  Ранний альфа-срез проекта.

- `ALPHA_improvements`  
  Последняя ветка, где ещё можно было управлять юнитами вручную.

- `ALPHA_vis`  
  Визуальные и presentation-эксперименты на базе ранней альфы.

## Важно про мультиплеер

В `cleanup/single-yagames-release` основной фокус смещён в single-player и Yandex-ready build.

Во многих других ветках репозитория всё ещё есть мультиплеер, сетевой код, серверная логика и разные стадии authority / lobby / VPS-экспериментов.

## Как запустить

Проект нужно открывать по HTTP, а не через `file://`.

### Мультиплеер / локальный сервер

```bash
npm install
npm start
```

Потом открой `http://localhost:3040`.

### Игра по интернету через tunnel

```bash
npm install
npm run tunnel
```

### Только одиночка как статика

```bash
npx --yes serve
# или
python -m http.server 8080
```

Потом открой адрес, который покажет сервер.

## Сборки

### Обычный single-player release

```bash
npm run build:release
```

### Архив для Яндекс Игр

```bash
npm run build:yandex
```

Архив появится в `dist/voidfront-yandex-upload.zip`.

## Тесты

```bash
npm test
```

## Публичность репозитория

Если репозиторий на GitHub публичный, люди смогут:

- открыть страницу проекта
- смотреть файлы и историю
- скачивать ZIP архива через GitHub
- клонировать репозиторий через `git clone`
- переключаться между ветками

## Примечание

В репозитории сохранена история нескольких направлений развития проекта: от ранних альфа-версий и ручного управления юнитами до мультиплеерных и server-authority веток, а также отдельной single-player ветки для Яндекс Игр.
