# VoidFront

Браузерная sci-fi RTS / tactical arena game.

Текущая основная ветка репозитория: `cleanup/single-yagames-release`.
Это самая свежая ветка под одиночный релиз и Яндекс Игры.

## Что в репозитории

- `cleanup/single-yagames-release`  
  Актуальная ветка под single-player релиз, чистку legacy-кода, ботов и подготовку билда для Яндекс Игр.

- `master`  
  Историческая основная ветка проекта до переноса фокуса на single-player release.

- `dev`  
  Общая рабочая ветка разработки до поздних релизных чисток.

- `dev-network`  
  Сетевые и мультиплеерные эксперименты.

- `dev-network-refactor`  
  Более глубокий рефактор сетевого слоя и authority-пайплайна.

- `local-zbt-4p-core`  
  Локальное ядро под 4 игроков и связанные боевые/коридорные эксперименты.

- `refactor/io-release-foundation`  
  Базовая ветка под релизный рефактор и подготовку структуры проекта.

- `new-design`  
  Ветка с UI/UX и визуальными редизайн-экспериментами.

- `VPS-release-v0.1`  
  Подготовка self-hosted / VPS-мультиплеерного релиза.

- `COSMIC_REBALANCE`  
  Отдельная ветка под ребаланс экономики, боёвки и темпа матча.

- `ALPHA`  
  Ранний альфа-срез проекта.

- `ALPHA_improvements`  
  Последняя ветка, где ещё можно было управлять юнитами вручную.

- `ALPHA_vis`  
  Визуальные и presentation-эксперименты на базе ранней альфы.

## Важно про мультиплеер

В `cleanup/single-yagames-release` основной фокус смещён в single-player и Yandex-ready build.

Во многих других ветках репозитория всё ещё есть мультиплеер, сетевой код, серверная логика и разные стадии authority / lobby / VPS-экспериментов.

## Как запустить

Проект нужно открывать по HTTP, а не через `file://`.

### Полная установка с нуля для Windows

Если у человека вообще ничего не стоит, вот самый простой путь.

1. Установить `Node.js`
   Скачай LTS-версию с [nodejs.org](https://nodejs.org/).
   Во время установки просто жми `Next` и оставь включённым добавление в `PATH`.

2. Перезапустить терминал
   После установки закрой PowerShell / CMD / Windows Terminal и открой заново.

3. Проверить, что всё встало

   ```bash
   node -v
   npm -v
   ```

   Если обе команды показывают версии, значит всё нормально.

4. Скачать проект

   Вариант A, проще всего без `git`:
   - открыть страницу репозитория на GitHub
   - нажать `Code`
   - нажать `Download ZIP`
   - распаковать архив в любую папку

   Вариант B, если нужен `git` и переключение между ветками:
   - установить Git for Windows с [git-scm.com](https://git-scm.com/download/win)
   - после установки выполнить:

   ```bash
   git clone https://github.com/PizzaGqD/VoidFront.git
   cd VoidFront
   ```

5. Открыть папку проекта в терминале

   Если проект скачан ZIP-архивом:
   - зайди в распакованную папку
   - кликни по адресной строке в проводнике
   - введи `powershell` или `cmd`
   - нажми Enter

6. Установить зависимости

   ```bash
   npm install
   ```

   Это нужно сделать один раз после скачивания проекта или после обновления `package.json`.

7. Запустить проект

   Для локального сервера:

   ```bash
   npm start
   ```

   Потом открыть в браузере:

   ```text
   http://localhost:3040
   ```

8. Если нужна только одиночка как статика

   ```bash
   npx --yes serve
   ```

   Или:

   ```bash
   python -m http.server 8080
   ```

   После этого открыть адрес, который покажет терминал.

9. Если команды не работают
   - если `node` или `npm` не найдены, значит Node.js не добавился в `PATH`; обычно помогает перезапуск терминала или перезагрузка ПК
   - если `git` не найден, значит Git не установлен; можно просто использовать `Download ZIP`
   - если PowerShell ругается на выполнение скриптов, открой обычный `cmd` и запускай команды там

10. Если человек вообще не хочет ничего ставить кроме Node.js
   Достаточно:
   - установить `Node.js`
   - скачать `ZIP` с GitHub
   - распаковать
   - открыть терминал в папке
   - выполнить `npm install`
   - выполнить `npm start`

### Мультиплеер / локальный сервер

```bash
npm install
npm start
```

Потом открой `http://localhost:3040`.

### Игра по интернету через tunnel

```bash
npm install
npm run tunnel
```

### Только одиночка как статика

```bash
npx --yes serve
# или
python -m http.server 8080
```

Потом открой адрес, который покажет сервер.

## Сборки

### Обычный single-player release

```bash
npm run build:release
```

### Архив для Яндекс Игр

```bash
npm run build:yandex
```

Архив появится в `dist/voidfront-yandex-upload.zip`.

## Тесты

```bash
npm test
```
